# employee manage system

### 1.登录模块，分为管理员与普通员工登录

### 2.员工信息管理模块，这里是核心模块，其中的员工ID直接影响后续的操作

### 3.考勤，包括依据员工ID来查找考勤记录，考勤记录分为3类，日常打卡，请假，加班

### 4.薪资管理模块，主要包括员工薪资的增删改查，自动计算员工的薪资并展示，这里的计算薪资会显示到员工信息管理模块中的薪资

### 5.部门与岗位模块，增删改查，加载时会检索核心模块的相关信息，并同步到该模块的文本文件

### 6.工具模块，封装了一些基本工具，比如增删改查工具
